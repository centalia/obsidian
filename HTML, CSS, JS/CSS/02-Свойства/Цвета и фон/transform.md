img {

## Масштаб: scale

| Свойство      | Эффект                   |
| ------------- | ------------------------ |
| `scale(1.2)`  | Увеличение по обеим осям |
| `scaleX(0.5)` | Сжатие по X              |
| `scaleY(2)`   | Увеличение по Y          |

```css
img {
  transform: scaleX(0.5);
}
```
