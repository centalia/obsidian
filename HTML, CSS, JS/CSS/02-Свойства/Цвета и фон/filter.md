
---

| Фильтр              | Описание                            |
|---------------------|-------------------------------------|
| `blur(2px)`         | Размытие                            |
| `brightness(120%)`  | Яркость                             |
| `saturate(150%)`    | Насыщенность                        |
| `contrast(110%)`    | Контраст                            |
| `invert(100%)`      | Инверсия цветов                     |

```css
img {
  filter: brightness(120%) blur(2px);
}
```
